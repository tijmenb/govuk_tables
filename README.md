# GOV.UK Tables

Build tables in Rails using the GOV.UK Design System.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'govuk_tables'
```

## Usage

The simplest example:

```erb
# app/views/index.html.erb

```
